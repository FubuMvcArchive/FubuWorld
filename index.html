The world of Fubu

A brief outline of the Fubu ecosystem.

Foundational

  • FubuCore

    A foundational library for .Net frameworks including model binding and localization

  • Bottles

    Shared libraries for runtime and deployment packaging of .Net applications

  • FubuLocalization

    Compact core library for Localization

  • StructureMap

    The Original Dependency Injection/Inversion of Control tool for .NET

 

Web Development

  • FubuMVC

    The MVC framwork that gets out of your way

  • HtmlTags

    NET objects for generating HTML
